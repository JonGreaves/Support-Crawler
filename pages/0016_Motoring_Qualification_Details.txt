You can add and edit details relating to any motoring qualifications that the driver has obtained.
Description
Select the description for the motoring qualification from the drop-down list.
Values: ABI code list 38.
Date Passed
Type the date the driver passed their qualification. You cannot enter a future date in this field.
Format: DD/MM/YYYY
