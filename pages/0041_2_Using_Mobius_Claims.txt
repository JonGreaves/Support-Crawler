The Dashboard Page
Using your internet browser, log in to the Claims Cloud. For details, see
Logging in to Mobius Claims
. The
Dashboard
page is displayed.
To return to the Dashboard page at any time after navigating away, click the banner title on the far left of the Top Navigation Bar. By default this banner is named 'THE CLAIMS CLOUD'. To edit the banner title, see
2: Using Mobius Claims
.
For details on the Top Navigation Bar, see
The Top Navigation Bar
.
Note:
To edit the default banner title:
Select
Configuration
,
Parameters
.
Optionally select
Group
from the Filter drop-down list.
Locate the
Banner Title
setting.
Click the
Edit
button on the right, and then enter the banner text in the text box.
Click
Save
to change the banner text, or click
Cancel
to discard any changes.
For more details, see
The Parameters Page
.
