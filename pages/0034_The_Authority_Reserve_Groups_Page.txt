The Authority Reserve Groups Page
Using your internet browser, log in to
Mobius Claims
. For details, see
Logging in to Mobius Claims
. The Dashboard page is displayed.
If you have already logged in, use the direct link in the Top Navigation Bar by clicking the
CONFIGURATION
item on the right. For details on the Top Navigation Bar, see
The Top Navigation Bar
.
The Configuration page provides a central location to access software configuration settings.
Click
AUTHORITY RESERVE GROUPS
to open the 'Authority Reserve Groups' page.
The Authority Reserve Groups page enables you to create maintain various different reserve groups. Grouping users in this way is the quickest and most transparent way to administer user rights.
To
ADD
an Authority Reserve Group, see
Adding Authority Reserve Groups
To
EDIT
an Authority Reserve Group, see
Editing Authority Reserve Groups
To
REMOVE
an Authority Reserve Group, see
Removing Authority Reserve Groups
Note: To return from one of the option pages to the Configuration page, select the
CONFIGURATION
option from the Top Navigation Bar.

Editing Authority Reserve Groups
On the Authority Reserve Groups page, locate the reserve group you want to edit, and then select the relevant
EDIT
button on the right. This opens the 'Edit <GROUP_NAME> Group', where <GROUP_NAME> is the group selected.

Removing Authority Reserve Groups
On the Authority Reserve Groups page, locate the reserve group you want to remove, and then select the relevant
EDIT
button on the right. This opens the 'Edit <GROUP_NAME> Group', where <GROUP_NAME> is the group selected.
Click the
REMOVE GROUP
button to delete the selected group. The following message is displayed:
'The authority reserve group will be removed. Are you sure?'.
Click
YES
to delete the selected group. This returns you to the 'Edit Group' page. Alternatively, click
NO
to keep the group. This returns you to the 'Authority Reserve Groups' page.
Note: To return from one of the option pages to the Administration page, select the
CONFIGURATION
option from the Top Navigation Bar.
