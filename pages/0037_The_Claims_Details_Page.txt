The Claims Details Page
The Claims Details page enables you to enter all the details required to start the claims process. You can access the
Claim Summary
page in various ways:
Click the
LOAD CLAIM
button on the
Dashboard
page
See
2: Using Mobius Claims
Click the
LOAD CLAIM
button on the
Diary
page
See
The Diary Page
Click the claim wanted from the
Claim Search Results
pane
See
The Search Page
If the
Claims Details
page has already been completed, the Claims Summary page is loaded instead. See
The Summary Page
.

The Claims Details pane
(LOB specific)
Clicking
NEXT
at the bottom of this pane, takes you to the following pane.
You can also navigate the different middle panes by using the shortcuts in the
Motor Claims
pane on the left. The pane used in this example is LOB dependent.
Shortcuts that have a green background indicate that the pane has been completed.
Shortcuts that have a red background indicate that the pane has not been completed.
Hovering over any of the shortcuts will highlight the colour and display a white tick
for completed or a white cross
for not completed.
The Notes pane
This pane lists the latest added notes.
The
ADD
button enables you to add a note to the Diary. The VIEW ALL button has the total number of notes in brackets Claim Number.

The Claim Number pane
This pane shows the Claim Number, the status ('Open' for example), and who the claim is currently assigned to.
The Script pane
This pane displays instructions for the claims handler depending on the claims question you are completing.
The text for the relevant scripts is defined in the Scheme.
