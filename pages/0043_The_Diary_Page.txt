Diary Search
Click the
DIARY SEARCH
button to open the
Diary Search
page, which enables you to run a very detailed search of actions, events and notifications.
Fields
You can only edit the following fields if you first tick the relevant checkboxes on the left.
Reminder Date
(from) - select the appropriate date and time from the date picker. See
Using the Date Picker
.

- As above.

(from) - select the appropriate date and time from the date picker. See
Using the Date Picker
.
to
- As above.
Created Date
(from) - select the appropriate date and time from the date picker. See
Using the Date Picker
.
to
-  As above.
Created By
- select the appropriate person or task from the drop-down list .
Owner User
- select the appropriate person or task from the drop-down list.
Owner Workgroup
- select the appropriate Workgroup from the drop-down list.
Diary Type
- select the appropriate Diary Type from the drop-down list.
Completion Type
- select the appropriate Completion Type from the drop-down list.
Claim Number
- enter the relevant Claim Number if available.
Click the
SEARCH
button to start searching the Diary. Click
Back
to abandon the search.
Note:
If you tick a checkbox, but you do not complete the relevant field, the message 'Field required' is shown. Complete the field, or untick the checkbox in question.
If a search is successful, the
Diary Search
page lists all matching claims.

Searching Diary Entries Again
Click the
SEARCH AGAIN
button to run another search. For instance to immediately verify if a diary entry has been reallocated.
Re-allocating Diary Entries
Select the one or more Diary entries.
Select the
Workgroup
or
User
radio button to for the kind of re-allocation.
Select the actual workgroup or user wanted from the relevant drop-down list.
Click the
RE-ALLOCATE
button .
