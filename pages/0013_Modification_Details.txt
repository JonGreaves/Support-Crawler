Modification Details screen
The Modifications Details screen follows the Vehicle Details screen in the initial data capture for a new prospect and allows you to enter details of any vehicle modifications.
This screen is displayed when you click the
button in the Modifications screen. See
Modifications screen
for more details on working in that screen.
Type
Select the modification from the drop-down list.
Values: ABI code list 86
Details
Type a short description of the modification details.
Value
Type the value of the modifications in GBP.
For Medical Reasons
Tick the check box if the modifications were made for medical reasons, untick if not.
Insurer Notified
Tick the check box if the insurer was notified, untick if not.
