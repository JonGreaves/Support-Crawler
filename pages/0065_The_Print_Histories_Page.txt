The Print Histories Page
Using your internet browser, log in to
Mobius Claims
. For details, see
Logging in to Mobius Claims
. The Dashboard page is displayed.
If you have already logged in, but are no longer on the Administration page, use the direct link in the Top Navigation Bar by clicking the
ADMINISTRATION
item on the left. For details on the Top Navigation Bar, see
The Top Navigation Bar
.
The Administration page provides a central location to access software related settings.
Click
PRINT HISTORIES
to open the
Print Histories
page.
Select the User who's Print History you want to review from the drop-down list.
Click the preview icon
for any given document to view its contents in a Preview pop-up screen.
When hovering over the preview page, document related information is displayed, and several document related options become available.
