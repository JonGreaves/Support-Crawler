to open the
Permission Groups
page.
This page enables you to link tasks with users.

Adding Permission Groups

Adding Permission Groups
Click the
ADD NEW GROUP
button to open the
Create 'New Group' Permissions
page.
This page has the following tabs:
Settings
- see
The Settings Tab
Tasks
- see
The Tasks Tab
Users
- see
The Users Tab
The Settings Tab
Click the
Settings
tab to enter basic details.
Fields
Role Level for group
- select the role level ('Administrator', 'Agent Administrator', or 'User') from the drop-down list, the Role level determines which kind of tasks will be available in the Task tab.
Permission group name
- enter the new group name
Note: If you want to also complete any other tabs, it is quicker to do this first and save your changes after; saving your changes now will return you to the previous page.
Click the
SAVE CHANGES
button to secure all changes. Alternatively, click
CANCEL
to discard any changes. Clicking either of these buttons produces a pop up window with a choice of
YES
to confirm your selection or
NO
to cancel.
