The Parameters Page
Using your internet browser, log in to
Mobius Claims
. For details, see
Logging in to Mobius Claims
. The Dashboard page is displayed.
If you have already logged in, use the direct link in the Top Navigation Bar by clicking the
CONFIGURATION
item on the right. For details on the Top Navigation Bar, see
The Top Navigation Bar
. The Configuration page provides a central location to access software configuration settings.
Click
PARAMETERS
to open the 'Parameters' page, subject to having access rights.
Parameter names are listed by age (in a non-alphabetical order).
Use
CTRL
+
F
to search.
Columns
Name
- Short descriptive name for the parameter
Description
- Explanatory details regarding the parameter function
Value
- Current value for this parameter
Group
- You can filter the table contents by group by selecting a given parameter group from the
Group
Filter drop-down list. For a list of the default parameter groups, see
Parameter Groups
.
Type
- You can filter the table contents by parameter type by selecting a given type from the
Type
Filter drop-down list. The types are:
Boolean
- true/false (yes/no)
Integer
- a numerical value
String
- one or more characters
To remove a filter, select Filter from the drop-down list.
Buttons
EDIT
- see
Editing Parameters
.
RESTORE DEFAULT
- see
Restoring Default Parameters
.
Editing Parameters
Some parameters are listed as 'Not Configurable'. This may be due to your access rights.
To amend a parameter, first click the
EDIT
button. Then, amend any of the values in the different columns. See
Columns
. You may not have rights to edit all parameters listed.
Click the
SAVE
button to secure all changes. Alternatively, click
CANCEL
to discard any changes.
Restoring Default Parameters
Click the
RESTORE DEFAULT
button to  roll back to the default values. This is useful if you do not recall the default parameters or if you have made changes in error.
