Medical Conditions screen
This screen is displayed when you click
or
in the Health Conditions screen. See
Health Conditions screen
for more information on working in that screen.
During a quote, you can enter any medical conditions that have to be disclosed for motor insurance purposes.
Description
Select the appropriate description for the medical condition from the drop-down list.
Values:  ABI code list 3
DVLA notified
Tick if you have advised the DVLA of the driver's condition, untick if not.
Date Diagnosed
Type the date the driver's medical condition was first diagnosed.
Licence Issued
Tick if a licence was issued.
Expiry Date
Type the licence expiry date.
Format: DD/MM/YYYY
After you save the record, you can add further medical conditions using this screen.
