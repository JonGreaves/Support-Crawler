Drivers screen
This screen is not part of the initial data capture. However, once a quote has been prepared (or the Summary Screen has been displayed at the end of data capture just prior to the first quote being generated) this screen is shown if either the
Named Drivers
menu option or
Drivers
button is selected. See
4: 4: Summary Screen
for more information on working within the Summary Screen.
The Drivers screen allows the relevant driver to be selected (if amendments are required) and also allows new drivers to be added or existing drivers to be removed.
