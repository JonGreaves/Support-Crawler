Immediate Tasks
The
Custom Actions to Trigger
drop-down list enables you to add an action for the given task. First click the Plus
button to add an action .
The
Policy System Events
drop-down list enables you to add an event for the given task. First click the Plus
button to add an event.
Select an action from the drop-down list, and then click the tick button
to save the action or the event to the selected task, or click the delete
icon to discard the new action or event.
Saved actions and events can be edited and deleted.
To edit saved actions and events, click the edit
icon to make any changes, click the delete
icon to remove the action or event.

Custom Workflow Categories
Categories in the Custom Workflow Categories pane are all added and amended in the same way. You cannot delete any custom workflow categories.
To add a new custom category, see
Adding a New Custom Category
To edit a custom category, see
Editing a Custom Workflow Category
Adding a New Custom Category
Click the
ADD NEW CATEGORY
button to enter the details for a new Custom Workflow Category. This adds a blank line at the bottom of the table.
Enter a meaningful category name. You now have the following options:
Click
SAVE
to secure your input. The following message is briefly displayed on the bottom right of the page: 'Item saved'.
Click
CANCEL
to discard the new Diary Type. The 'new' line may not be cleared but it will not appear again after rebuilding the page.

Rules
Rules influence the workflow and should only be added and applied after due consideration.
If the Rules checkbox is unticked, then the workflow will always continue.
If unticked, tick the Rules checkbox to open the Rules pane.
Click the
+ ADD RULE
button to add a line for a Rule.
Model Name
- select the Model to be used
Conditions
select the condition from the drop-down list
select the operator from the drop-down list
= (equals)
!= (does not equal)
< (is smaller than)
> (is bigger than)
<= (is smaller than or equal)
=> (is bigger than or equals)
has changed (time span required)
enter the condition, or select the condition from the drop-down list
Click the tick button
to save the Rule to the selected Workflow Category. Alternatively, click the X button
to discard the new Rule.
If required, click the
+ ADD CONDITION
button to add a Condition line for this Rule.
Adding a Condition line also adds a
- REMOVE
button for every condition.
Adding a Condition line also adds an '
and
/
or
' drop-down field in the first Condition line. The '
and
/
or
' Condition applies to all conditions within a Rule.
Click the tick button
to save the Rule to the selected Workflow Category, or click the X button
to discard the new Rule.
