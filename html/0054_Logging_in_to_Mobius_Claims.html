Logging in to Mobius Claims

Open you internet browser and navigate to https://<your_site>.theinsurercloud.com, where <your_site> will have been defined when Mobius Claims was configured. This opens the 'Welcome' page.

With the exception of the banner and About (see About), clicking any of the menu options when you are not logged in, opens the Login pane on the 'Mobius Login Server' page.

Log in Options

  • On the left you have to option to log in locally , see Local login

  • On the right you have the option to log in through Active Directory (AD), see Active Directory login.

Note:

To edit the default banner title:

  1. Select Configuration, Parameters.

  2. Optionally select Group from the Filter drop-down list.

  3. Locate the Banner Title setting.

  4. Click the Edit button on the right, and then enter the banner text in the text box.

  5. Click Save to change the banner text, or click Cancel to discard any changes.

For more details, see The Parameters Page.

Local login

  1. In the Local Login pane, complete the following fields. Username - Enter your Claims Cloud user name. Password - Enter your Claims Cloud password

  2. Optionally, tick the Remember My Login check box.

  3. Click Login when done. This opens the Dashboard. See 2: Using Mobius Claims.
    Alternatively, click Cancel to abandon logging in.

Active Directory login

  1. In the Active Directory pane, select the appropriate Active Directory (AD) account.

    • Mobius, see Mobius AD Sign In

    • Corporate, see Corporate

  1. Optionally, tick the Keep me signed in check box. Only tick the Keep me signed incheck box if you are working on your own device; doing so on a 'public' device will allow others to log in using your account.

  1. Click Log in when done. This opens the Dashboard. See 2: Using Mobius Claims. Alternatively, click Cancel to abandon logging in.

Mobius AD Sign In

  1. Enter your Organizational Account login name.

  2. Enter your Organizational Account password.

  3. Click Sign in when done. This opens the Claims Cloud Dashboard page. See 2: Using Mobius Claims. Alternatively, click Cancel to abandon logging in.

Corporate

  • Your Microsoft Login Account, see Your Microsoft Login Account
