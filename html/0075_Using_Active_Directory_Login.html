Using Active Directory Login

What's Covered in this topic?

  • Using Active Directory Login
    • Active Directory login
    • Mobius AD Sign In
    • Corporate

Start your internet browser and navigate to https://<your_site>.theinsurercloud.com, where <your_site> will have been defined when the Claims Cloud was configured.

Active Directory login

  1. In the Active Directory pane, select the appropriate Active Directory (AD) account.

    • Mobius, see Mobius AD Sign In

    • Corporate, see Corporate

  1. Optionally, tick the Keep me signed incheck box.

Only tick the Keep me signed incheck box if you are working on your own device; doing so on a 'public' device will allow others to log in using your account.

  1. Click Log in when done. This opens the Dashboard. See 2: Using Mobius Claims

Mobius AD Sign In

  1. Enter your Organizational Account login name.

  2. Enter your Organizational Account password.

  3. Click Sign in when done. This opens the Claims Cloud Dashboard page. See 2: Using Mobius Claims

Corporate

Select one of the following login methods:

  • Your Microsoft Login Account, see Your Microsoft login account

  • Use Another Account, see Use another account
