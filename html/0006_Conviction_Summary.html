Conviction Summary screen

This screen enables you to add or maintain driver convictions.

It follows the Driver screen in the initial data capture for a new prospect; see Driver Information for more information on working within that screen.

Use the button to add convictions.

Use the and buttons to maintain existing convictions.

Existing convictions are edited in the Conviction Details screen. See Conviction Details screen for guidance on working within this screen.

After you save the policy or prospect record, you can open the Conviction Summary screen from the Summary Screen. See 4: 4: Summary Screen for guidance on working in this screen.
