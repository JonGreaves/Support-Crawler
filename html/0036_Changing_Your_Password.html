Changing Your Password

This topic details how to change your own user password.

  • For details on how to change the password for other users, (subject to having Administrator rights), see the Admin Panel.

  • For details on setting password requirements (subject to having Administrator rights), see The Parameters Page.

To change your password for Mobius Claims:

  • Log in to Mobius Claims. For guidance, see Logging in to Mobius Claims.

  • Click the Hi <User_Name> drop-down list in the top right of the page, and then select Change password; this opens the Change Password pane

  • Enter your Old Password for security purposes, and then enter your new password twice; once in the New Password field and once in the Confirm Password field.

    Click CHANGE PASSWORD to apply the new password. Alternatively, click CLEAN FIELDS to discard any changes.

  • If the password change is successful, a brief pop-up message in the bottom right of the page displays 'Password Updated'.

  • If the old password is incorrect, a brief pop-up message at the bottom right of the page displays the message 'Unable to update password / Incorrect Username or Password'.

  • If the new passwords are not identical the error 'Passwords do not match' is displayed.

An Administrator can configure security settings, such as password age and complexity, for all users by selecting Configuration, and then Parameters. See Password.

Note:

It is good business practice to:

  • change any password assigned to you to a password that only you know

  • change any password at least every 90 days

  • For security purposes passwords should minimally contain:

    • six characters

    • one upper case (capital) letter

    • one lower case letter

    • one digit (numeral)

    • one special character (!, ?, %, and so on)
