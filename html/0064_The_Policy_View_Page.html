The Policy View Page

You can access the Policy View page in the following ways:

  • Click POLICY VIEW in the Navigation Pane when creating a claim, see The Navigation Pane

  • Click POLICY VIEW in the Navigation Pane on the Summary page, see The Navigation Pane (upper left)

The Policy View page

This page displays any configured information (configured in the CSV files, like the FNOL fields) in collapsible panels and nothing is editable. The purpose of the Policy View page is to display the configured policy information using the history of the policy at the date and time of the incident (as captured in the Incident Date question during FNOL).

Example: If the Incident Date question is captured as 11/06/2019, and the policy has an MTA on 18/06/2019, then the MTA information is not displayed because it had not been applied to the policy at the date and time of the incident. This also applies to any configured FNOL fields if these have been pre-populated from Mobius Claims.

Tabs

The Policy View page has the following tabs:

  • Summary - see The Summary tab

  • Excesses - see The Excesses tab

  • Endorsements - see The Endorsements tab

  • Documents - see The Documents tab

  • Financials - see The Financials Tab

  • Loss Ratio - see The Loss Ratio Tab

The Navigation Pane (upper left)

The Navigation pane enables you to add new claims and access existing claims.

Click < ADD NEW CLAIM to open the Action pane, see The Action Pane.

The Claims Details Pane (middle left)

Select any of the red or green links to open the Claims Details page.

This pane is Line of Business dependent.

Click EXISTING CLAIMS to access existing claims, see Policy Search Results.

The Notes Pane (lower left)

This pane lists the latest added notes. The ADD button enables you to add a note to the claim. The VIEW ALL button shows the number of notes for the selected Claim in brackets.

  • Click ADD to open the Add Note popup window. You must select the Type of note from the 'Select type' drop-down list before submitting the new note. Enter the contents of the note in the text box below. Click OK to save any changes. Alternatively click the Close button in the top right of the window to discard the new note.

  • Click VIEW ALL to open the View Notes popup window. Use the scroll bar on the right to view all notes if necessary. To close the popup window, click the Close button in the top right of the window.

The Claim Number Pane (upper right)

This pane shows the Claim Number, the status ('Open' for example), and who the claim is currently assigned to.

When you select the Summary tab on the Policy View page, all panes on this tab are automatically expanded. Click any blue bar to toggle between collapsing and expanding the relevant pane. All fields are informational only. There is no option to dig down on fields.

The Documents tab

Click any of the light blue bars to toggle between collapsing and expanding the relevant pane per policy history. For example, from 'Start Date' to 'First MTA', and then from 'MTA' to 'End Date'.

The Preview button opens a pop up window which is identical to the Correspondence preview popup, showing the document content. See The Correspondence Tab.

All fields are informational only. There is no option to dig down on fields.

Click the PREVIEW button to view the selected document.

The Financials Tab

The financials tab enables you to view all Open Transactions and all Settled Transactions for the selected policy.

All fields are informational only. There is no option to dig down on fields.

The Loss Ratio Tab

The Loss Ratio tab enables you to view the loss Ratio percentage on the basis of the Policy Premium paid set of against the Claim Value.

Click the down-arrow q at the end of the first line to view the ratio per year.

Click the down-arrow q at the end of the first line to view the ratio per claim for the selected year.

All fields are informational only.

Click the up-arrow p at the end of the line to collapse an expanded line.
